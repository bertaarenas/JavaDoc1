# JavaDoc1
Este es una programa sencillo donde se simula un inicio de sessión.
El código utiliza arrays para almazenar los usuarios y contraseñas correspondientes, y un while para tener intentos infinitos.
El programa solicita al usuario el usuario y contraseña y compara los datos. Si los datos cinciden el programa termina. En caso de que no, con el try catch se lanzará una excepción, se mostrará un error y se volverá a pedir el usuario y contraseña.
